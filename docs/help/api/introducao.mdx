# Introdução à API

A API REST do Manager permite integrar sistemas externos, automatizar fluxos e construir ferramentas sobre a plataforma.

## Autenticação

Todos os endpoints (exceto `/api/auth/login`) requerem um token JWT:

```
Authorization: Bearer <token>
```

Obtenha o token via `POST /api/auth/login` com email e senha.
