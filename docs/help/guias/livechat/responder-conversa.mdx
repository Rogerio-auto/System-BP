# Responder conversas

O painel de conversa exibe o histórico completo de mensagens com o cliente e um compositor para enviar respostas. Este guia explica como navegar pelo painel, responder mensagens e lidar com a janela de 24h do WhatsApp.

<Callout type="info" title="Pré-requisito">
  Você precisa ter o papel de **atendente** ou superior e ao menos um canal conectado (WhatsApp Business, Instagram Direct ou WAHA) para enviar mensagens.
</Callout>

## O que é o painel de conversa

O painel de conversa é dividido em duas áreas:

- **Lista de mensagens** — exibe todo o histórico da conversa em ordem cronológica. Mensagens enviadas pelo cliente aparecem à esquerda; mensagens enviadas pela equipe ou pela Ana Clara aparecem à direita.
- **Compositor** — área na parte inferior para digitar e enviar novas mensagens.

### Tipos de mensagem suportados

O painel exibe automaticamente todos os tipos de mensagem do WhatsApp:

| Tipo | O que exibe |
|---|---|
| Texto | Conteúdo em texto simples |
| Imagem | Preview da imagem com link para ampliar |
| Vídeo | Miniatura com botão de play |
| Áudio / Voz | Player de áudio nativo |
| Documento | Nome do arquivo e botão de download |
| Sticker | Preview da figurinha |
| Localização | Coordenadas com link para Google Maps |
| Contato | Nome do contato compartilhado |
| Template | Nome do template e corpo renderizado |
| Interativo | Botões / lista de opções (somente leitura) |
| Story / Comentário | Conteúdo somente leitura (sem resposta direta) |

### Status das mensagens enviadas

Mensagens enviadas pela equipe exibem um ícone de status ao lado do horário:

- **1 check cinza** — Mensagem enviada ao servidor
- **2 checks cinzas** — Entregue ao dispositivo do cliente
- **2 checks azuis** — Lida pelo cliente
- **X vermelho** — Falha no envio (tente novamente)

## Como responder uma mensagem

<Step number={1} title="Abra a conversa no inbox">
  Na lista lateral do Live Chat, clique na conversa que deseja responder. O painel de conversa abre à direita com o histórico carregado.
</Step>

<Step number={2} title="Leia o histórico">
  Role para cima para ver mensagens anteriores. O sistema carrega automaticamente mensagens mais antigas ao chegar no topo da lista.
</Step>

<Step number={3} title="Digite a mensagem no compositor">
  Clique na área de texto na parte inferior e digite a resposta. O campo cresce automaticamente até 4 linhas conforme você digita.
</Step>

<Step number={4} title="Envie a mensagem">
  Pressione **Ctrl+Enter** (Windows/Linux) ou **Cmd+Enter** (Mac) para enviar, ou clique no botão de envio (seta azul) no canto direito do compositor.

  <Callout type="info" title="Envio otimista">
    A mensagem aparece imediatamente na lista enquanto é processada pelo servidor. Se houver falha, ela é removida e você recebe uma notificação de erro.
  </Callout>
</Step>

## Janela de 24 horas (WhatsApp)

O WhatsApp Business permite enviar mensagens de texto livre apenas dentro de uma janela de **24 horas** após a última mensagem recebida do cliente.

### O que acontece quando a janela expira

Quando a janela de 24h expira, o compositor exibe um aviso amarelo:

> "A janela de 24h expirou. Use um template para retomar o contato."

Nesse estado:
- O campo de texto fica desabilitado
- O botão de envio fica inativo
- Um botão **Usar template** aparece no aviso

### Como retomar o contato após a janela expirar

<Step number={1} title="Clique em Usar template">
  No aviso amarelo da janela expirada, clique no botão **Usar template**.
</Step>

<Step number={2} title="Selecione o template aprovado">
  Escolha um dos templates aprovados pela Meta para o seu número de telefone. Templates precisam ser cadastrados previamente no painel da Meta.
</Step>

<Step number={3} title="Envie o template">
  Confirme o envio. Quando o cliente responder, a janela de 24h é reaberta e você pode voltar a enviar texto livre.
</Step>

<Callout type="warning" title="Limite da API do WhatsApp">
  A Meta não permite enviar texto livre fora da janela de 24h. Tentativas de envio sem template são bloqueadas automaticamente pela API. Use sempre templates aprovados para iniciar conversas proativamente.
</Callout>

## Enviar arquivos e mídias

Para enviar uma imagem, vídeo, áudio ou documento:

1. Clique no ícone de **clipe** no canto esquerdo do compositor
2. Selecione o arquivo no gerenciador de arquivos
3. Aguarde o upload e envie

<Callout type="info" title="Formatos suportados">
  Imagens (JPG, PNG, WebP), vídeos (MP4), áudios (MP3, OGG), documentos (PDF, DOC, DOCX). Tamanho máximo: conforme limites do WhatsApp Business API (geralmente 16MB para mídia).
</Callout>

## Histórico longo e carregamento de mensagens antigas

O painel virtualiza a lista de mensagens para manter a performance mesmo com centenas de mensagens. Quando você chega ao topo:

- Um botão **Carregar mensagens anteriores** aparece
- Clique para buscar a próxima página de histórico

## Erros comuns

| Situação | Causa | O que fazer |
|---|---|---|
| Mensagem não enviada, notificação vermelha | Falha de rede ou API | Verifique a conexão e tente novamente |
| Compositor desabilitado sem aviso amarelo | Permissão insuficiente ou carregando | Aguarde alguns segundos ou atualize a página |
| Status trava em "Enviada" (1 check) | Cliente sem conexão ou número inválido | Normal — o status atualiza quando o dispositivo ficar online |
| Janela expirada imediatamente | Cliente nunca enviou mensagem primeiro | O WhatsApp exige que o cliente inicie o contato |
