# Pipeline MDX — referência de componentes

Esta página é uma referência viva dos componentes canônicos disponíveis em qualquer página de ajuda. Use como guia ao escrever conteúdo novo.

## Callouts

<Callout type="info">
  Use Callout `info` para informação contextual neutra.
</Callout>

<Callout type="tip" title="Dica do operador">
  Você pode dar um título customizado a qualquer Callout passando a prop `title`.
</Callout>

<Callout type="warn">
  Callout `warn` é o padrão para avisos sobre módulos em desenvolvimento.
</Callout>

<Callout type="danger">

</Callout>

## Steps

Sequência de passos numerados — padrão Stripe Docs:

<Step number={1} title="Abra o CRM">
  Clique no item **CRM** na sidebar para ver a lista de leads atribuídos a você.
</Step>

<Step number={2} title="Selecione um lead">
  Use a busca ou role até encontrar o lead. Clique no nome para abrir o detalhe.
</Step>

<Step number={3} title="Mova o card no Kanban">
  Arraste o card entre as colunas conforme avança o atendimento. O sistema registra
  cada movimentação para auditoria.
</Step>

## CodeBlock — TypeScript

<CodeBlock lang="ts" title="Buscar leads por status">
{`import { api } from '@/lib/api';

export async function fetchLeads(status: 'new' | 'qualifying' | 'closed_won') {
  return api.get<LeadResponse[]>('/api/leads', {
    params: { status, limit: 25 },
  });
}`}
</CodeBlock>
