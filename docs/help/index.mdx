# Central de Ajuda

Bem-vindo. Esta é a central onde você encontra tudo o que precisa para operar o **Manager Banco do Povo** com confiança — do primeiro acesso ao detalhe técnico.

<Callout type="tip" title="Atalho rápido">
  Pressione **Ctrl+K** (ou **⌘K** no Mac) em qualquer tela do sistema para abrir a busca. É a forma
  mais rápida de chegar a uma página.
</Callout>

## O que existe aqui

**Começar** — trilhas de primeiro acesso por papel:

- [Começar como administrador](/ajuda/comecar/admin)
- [Começar como gestor](/ajuda/comecar/gestor)
- [Começar como agente](/ajuda/comecar/agente)

**Guias** — _em breve._ Passo-a-passo para tarefas concretas: criar um lead, mover um card no Kanban, configurar uma régua de cobrança, aprovar uma análise.

**Conceitos** — disponível agora. Explicações dos pilares que sustentam o sistema. Recomendamos a leitura inicial dos três abaixo.

**API** — _em breve._ Referência técnica para quem integra o Banco do Povo a outros sistemas.

## Por onde começar

Estes três conceitos respondem 80% das dúvidas que aparecem nas primeiras semanas de uso:

- [Papéis e cidades](/ajuda/conceitos/papeis-e-cidades) — o que cada papel pode fazer e por que você só vê o que vê.
- [LGPD no Banco do Povo](/ajuda/conceitos/lgpd) — como o sistema protege os dados dos atendidos e o que você não deve fazer.
- [Módulos liberados](/ajuda/conceitos/modulos-liberados) — por que algumas funcionalidades aparecem como "em desenvolvimento".
