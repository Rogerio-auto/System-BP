# Central de Ajuda

Bem-vindo à Central de Ajuda do **Manager Banco do Povo**. Aqui você encontra tudo o que precisa para operar o sistema com confiança — desde o primeiro acesso até consultas técnicas da API.

## O que você encontra aqui

- **Começar** — tutoriais por papel: administrador, gestor e operador.
- **Guias** — passo-a-passo de cada módulo: CRM, análise de crédito, follow-up, cobrança, templates.
- **Conceitos** — explicações transversais: papéis, escopo por cidade, política LGPD.
- **API** — referência técnica para integração: endpoints, autenticação, exemplos.
